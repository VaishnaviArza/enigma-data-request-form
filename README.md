Enigma Portal is a full-stack web application built for the Neural Plasticity and Neurorehabilitation Laboratory (NPNL) at USC Chan to streamline how researchers and collaborators request access to neuroimaging and behavioral datasets.

The platform replaces ad-hoc email workflows with a structured, auditable, and role-based system for submitting, reviewing, and managing data requests.

## Key Features

- Dynamic data request form with metrics, cohorts, and subcategories  
- Secure authentication with email-based login  
- Role-based access (Admins vs. Requesters)  
- Cloud-ready architecture (Firebase + API backend)

## Tech Stack

- **Frontend:** React + TypeScript  
- **Backend:** Python (Flask)  
- **Auth:** Firebase Authentication  
- **Database / Storage:** AWS S3  
- **Deployment:** Designed for cloud hosting (AWS)
